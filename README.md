# Landing Page Project

## Table of Contents
 
> four sections dynamic landing page 
> dynamically build navigation using appendChild, and innerHTML.
> add active class to the section in the viewport and remove it from the other sections
 by using the .getBoundingClientRect() method to detect section in the viewport
> Using addEventListener('click',....) to listen to the click event.
> Using preventDefault() as if there is a default event occurring we need to stop that.
> using scrollIntoView() to make a smooth scrolling behavior 
> when clicking the items in the navigation bar goes to the appropriate section
